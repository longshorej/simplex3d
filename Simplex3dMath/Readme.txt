Project documentation is available at:
http://code.google.com/p/simplex3d/wiki/MathLibrary


Before you begin, you need to have:
  - Java 1.6.0_u15 or higher.
  - Ant 1.7 or higher.
  - SCALA_HOME set to Scala 2.8.0.RC1 or higher.

To build the project:
  1) open a console,
  2) cd into the project directory,
  3) type: ant jar

To make a custom jar with selected modules:
  1) open a console,
  2) cd into the project directory,
  3) type: ant custom-jar
  4) follow the prompt.
